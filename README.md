# Manus 自动注册工具平台

一套专业的自动化批量账号注册工具平台，具备卡密授权管控、实时任务监控、智能反检测及完善的管理后台。

## 🎯 核心功能

### 1. 卡密验证系统
- **卡密激活**：用户使用前需输入卡密进行激活
- **卡密管理**：后台支持生成与管理卡密，可设置有效期及使用次数限制
- **自动通知**：新用户激活和卡密即将到期时自动发送通知

### 2. 单账号注册
- **一键注册**：支持输入邮箱、密码、邮箱API、手机号、短信API、YesCaptcha Key、邀请链接
- **实时日志**：通过 SSE 实时推送注册日志
- **智能指纹**：随机生成浏览器指纹，规避检测

### 3. 批量任务注册
- **邮箱导入**：支持批量导入邮箱列表（格式：`邮箱----邮箱API`）
- **并发控制**：可设置 1-50 并发数
- **多任务并行**：支持创建多个任务并行执行

### 4. 实时任务监控
- **进度追踪**：通过 SSE 实时推送任务进度、成功/失败数及详细日志
- **进度条可视化**：直观展示注册进度
- **任务管理**：支持查看任务列表、取消正在运行的任务

### 5. 注册结果导出
- **表格展示**：以表格形式展示注册成功的账号信息（邮箱、密码、Token）
- **CSV 导出**：支持导出为 CSV 文件进行后续处理

### 6. 代理 IP 配置
- **多提供商支持**：集成 Luminati、Oxylabs、SmartProxy、Custom 四大提供商
- **自动轮换**：支持 4 种轮换策略（轮询、随机、加权、自适应）
- **健康检查**：定期检查代理可用性，自动标记不健康代理

### 7. 智能反检测
- **随机浏览器指纹**：User-Agent、屏幕分辨率、语言等随机生成
- **行为模拟**：模拟真实用户行为
- **网络特征伪装**：隐藏自动化特征

### 8. 管理员后台
- **卡密管理**：生成、查看、编辑、删除卡密
- **用户管理**：查看所有用户的使用记录
- **数据统计**：系统运行统计（总注册数、成功率、任务数）
- **历史记录**：完整的操作历史和数据统计分析

## 🚀 快速开始

### 系统要求
- Node.js 18+
- MySQL 5.7+ 或 TiDB
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 安装

```bash
# 克隆项目
git clone <repository-url>
cd manus_register_tool_gui

# 安装依赖
pnpm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入您的配置

# 数据库迁移
pnpm db:push

# 启动开发服务器
pnpm dev
```

访问 `http://localhost:3000`

## 📖 使用指南

### 第一步：激活卡密

1. 访问首页，点击 **卡密激活**
2. 输入您的卡密码
3. 点击 **激活** 按钮
4. 激活成功后，您将获得完整的功能权限

### 第二步：单账号注册

1. 登录后进入 **单账号注册** 页面
2. 填写以下信息：
   - **邮箱**：要注册的邮箱地址
   - **密码**：账号密码
   - **邮箱 API**：邮箱收信接口地址
   - **手机号**（可选）：用于手机绑定
   - **短信 API**（可选）：短信接收接口地址
   - **YesCaptcha Key**：验证码解决方案密钥
   - **邀请链接**（可选）：邀请码或推荐链接
3. 点击 **开始注册**
4. 实时查看注册日志和进度

### 第三步：批量任务注册

1. 进入 **批量任务** 页面
2. 点击 **导入邮箱列表**
3. 上传 CSV 或 TXT 文件，格式为：`邮箱----邮箱API`
   ```
   user1@example.com----https://api.example.com/email
   user2@example.com----https://api.example.com/email
   ```
4. 设置任务参数：
   - **任务名称**：任务标识
   - **并发数**：1-50 之间的数值
   - **YesCaptcha Key**：验证码密钥
5. 点击 **创建任务**
6. 进入 **任务监控** 页面查看实时进度

### 第四步：查看结果和导出

1. 任务完成后，进入 **任务结果** 页面
2. 查看注册成功的账号信息
3. 点击 **导出 CSV** 下载结果文件

### 代理配置

1. 进入 **代理配置** 页面
2. 选择代理提供商
3. 输入 API 凭据（如适用）
4. 配置轮换策略和健康检查参数
5. 点击 **保存配置**

## 🔧 管理员功能

### 卡密管理

1. 进入 **管理后台** → **卡密管理**
2. 点击 **生成卡密**
3. 设置：
   - **生成数量**：一次生成的卡密数
   - **最大使用次数**：每张卡密可使用的次数
   - **过期日期**：卡密的有效期
4. 点击 **生成** 按钮
5. 复制生成的卡密分发给用户

### 用户管理

1. 进入 **管理后台** → **用户管理**
2. 查看所有用户的使用记录
3. 支持按用户 ID 或操作类型搜索
4. 点击 **导出 CSV** 下载用户记录

### 数据统计

1. 进入 **管理后台** → **统计分析**
2. 查看系统运行统计：
   - 总注册数
   - 成功率
   - 任务数
   - 激活卡密数
3. 查看详细的数据分析

## 🐛 故障排除

### 问题：注册失败率高

**解决方案：**
1. 检查代理质量
2. 调整轮换策略为 `加权` 或 `自适应`
3. 增加健康检查频率
4. 检查 Turnstile 验证码配置

### 问题：代理连接超时

**解决方案：**
1. 检查代理 API 配置
2. 增加超时时间
3. 切换代理提供商
4. 检查网络连接

### 问题：卡密验证失败

**解决方案：**
1. 检查卡密是否已过期
2. 检查卡密使用次数是否已用尽
3. 查看数据库中的卡密状态
4. 重新生成卡密
