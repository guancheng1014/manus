## 测试场景

### 场景 1: 邀请链接解析测试

**目标**: 验证邀请链接能否正确解析

**步骤**:
1. 进入"单账号注册"页面
2. 在"邀请链接"输入框中粘贴以下链接：
   ```
   https://manus.im/invitation/PYJYCLPOGGZF?utm_source=invitation&utm_medium=social&utm_campaign=copy_link
   ```
3. 观察"邀请码"输入框是否自动填充为 `PYJYCLPOGGZF`

**预期结果**:
- ✅ 邀请码自动填充
- ✅ 邀请码输入框显示为禁用状态
- ✅ 邀请链接中的 UTM 参数被正确解析

**验证代码**:
```javascript
// 在浏览器控制台中运行
const url = 'https://manus.im/invitation/PYJYCLPOGGZF?utm_source=invitation&utm_medium=social&utm_campaign=copy_link';
const match = url.match(/\/invitation\/([A-Z0-9]+)/);
console.log('邀请码:', match?.[1]); // 应该输出: PYJYCLPOGGZF
```

## 故障排除

### 问题 1: 邀请码未自动填充

**原因**: 邀请链接格式不正确

**解决方案**:
1. 检查邀请链接格式
2. 确保包含 `/invitation/` 路径
3. 确保邀请码为大写字母和数字

### 问题 2: 邮箱验证失败

**原因**: 邮箱 API 配置错误

**解决方案**:
1. 验证 API URL 正确性
2. 检查 API 认证信息
3. 查看 API 文档确认端点

### 问题 3: 短信验证失败

**原因**: 短信 API 配置错误

**解决方案**:
1. 验证 API URL 正确性
2. 检查手机号格式
3. 确认 API 配额充足

### 问题 4: 注册超时

**原因**: 网络连接缓慢或 API 响应慢

**解决方案**:
1. 检查网络连接
2. 增加超时时间
3. 使用代理加速
